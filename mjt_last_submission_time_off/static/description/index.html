Overview

  • Companies manage daily resource allocation to ensure their activity can run properly. Usually, companies require employees to submit their leave request a few days before the leave date, therefore they can adjust the resources arrangement. This add-on will enable your company to define time off submission deadline based on time off type.

Screenshots

Define your time off submission deadline in Time Off > Configuration > Time Off Types and fill in the field Last Submission Date based on options below:

  • Negative value (e.g., -7): requires employees to submit time off request at least 7 days before time off date

  • Positive value (e.g., 7): allows employees to submit time off request up to 7 days after time off date

  • 0 value: allows employees to submit time off anytime

Odoo will show the following error message when an employee tries to submit a time off request past the submission deadline, and the time off request cannot be submitted. This will be a restriction, so they need to submit their leave proposal earlier than the deadline.

To let user create and approve over due time off request, go to user settings and select Administrator for Last Submission Time Off.

Installation Dependencies

  • To install this module, you need to install Time Off module.
